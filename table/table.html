Призначення Назва тегу Короткий опис Стандарт
Структура документа. HTML. < ! > Визначає коментар, який ігнорується аналізатором HTML < HTML4 >
< !DOCTYPE > Оголошує тип і формат вмісту документа < HTML4 >
< BODY > Визначає початок основної частини сторінки < HTML4 >
< HEAD > Укладає в собі теґи, що містять невидиму інформацію про документ < HTML4 >
< HTML > Ідентифікує документ який містить елементи HTML < HTML4 >
Заголовки і назви. HTML. < H1 > Заголовок 1-го рівня < HTML4 >
< H2 > Заголовок 2-го рівня < HTML4 >
< H3 > Заголовок 3-го рівня < HTML4 >
< H4 > Заголовок 4-го рівня < HTML4 >
< H5 > Заголовок 5-го рівня < HTML4 >
< H6 > Заголовок 6-го рівня < HTML4 >
< TITLE > Назва документа, що показується в заголовку вікна браузера < HTML4 >
Абзаци та рядки. HTML. < BR > Вставка кінця рядка < HTML4 >
< CENTER > Вирівнює укладені в нього компоненти по центру < HTML4 >
< HR > Поміщає горизонтальну лінію на сторінку < HTML4 >
Стилі в тексті. HTML. < B > Робить текст жирним, де він доступний < HTML4 >
< I > Перетворює шрифт тексту в курсивний, де це можливо < HTML4 >
< PRE > Являє текст шрифтом фіксованої ширини < HTML4 >
< S > Виводить текст закресленим < HTML5 >
< STRIKE > Виводить текст закресленим < HTML4 >
< STRONG > Виділяє текст жирним шрифтом < HTML4 >
< U > Робить текст підкресленим < HTML4 >
Списки. HTML. < Ll > Позначає один елемент у нумерованому або маркірованому списку < HTML5 >
< OL > Перетворює рядки тексту з теґами в нумерований список < HTML4 >
< UL > Перетворює рядки тексту з теґами в маркований список < HTML4 >
Посилання в HTML. < А > Гіперпосилання; обов'язково здавати атрибути HREF або NAME < HTML4 >
< А HREF="url" > Гіперпосилання на елемент іншого документа < HTML4 >
< А NАМЕ="імя якоря" > Гіперпосилання на елемент цього ж документа < HTML4 >
